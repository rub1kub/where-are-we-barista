# КРОТ / C++ ядро расчёта

C++ ядро — это экспериментальная консольная версия расчётной части. Она не содержит интерфейса и 3D-визуализации. Основной MVP остаётся в веб-приложении.

Модуль нужен для технической демонстрации переносимости расчётной цепочки: журнал радиовысотомера и карта высот подаются в отдельную программу, а на выходе получается навигационная оценка без React, Three.js и браузера.

## Входы

- журнал радиовысотомера NMEA-0183 GGA;
- карта высот `data/dem-sample.json`;
- высота борта над уровнем моря, по умолчанию `1500 м`;
- диапазон перебора путевой скорости.

Поле высоты в GGA используется как расстояние до земли от радиовысотомера.

Профиль рельефа считается по формуле:

```text
высота рельефа = высота борта над уровнем моря - расстояние до земли
```

## Выходы

- `x_m` / `y_m` — локальные координаты на карте высот;
- широта и долгота, если координата попадает в карту;
- путевая скорость;
- направление / азимут;
- показатель совпадения профилей;
- ошибка профиля;
- доверие к расчёту;
- статус `VALID`, `DEGRADED`, `AMBIGUOUS`, `LOW_RELIEF` или `NO_FIX`;
- время расчёта.

В человекочитаемом выводе статусы печатаются по-русски: например, `VALID` как `МЕСТО НАЙДЕНО`.

## Данные

В `data/dem-sample.json` лежит контрольная карта высот, экспортированная из текущего сэмпла Copernicus DEM GLO-30 проекта. В `data/control-radio-altimeter.nmea` лежит контрольный журнал радиовысотомера для того же района.

Это не заказной журнал реального борта. Файлы нужны для воспроизводимой проверки отдельного C++ ядра.

## Сборка

```bash
cd cpp-core
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build
```

## Запуск

```bash
./build/krot_cpp \
  --dem data/dem-sample.json \
  --nmea data/control-radio-altimeter.nmea \
  --baro 1500 \
  --speed-min 35 \
  --speed-max 65 \
  --speed-step 1
```

JSON-вывод:

```bash
./build/krot_cpp --dem data/dem-sample.json --nmea data/control-radio-altimeter.nmea --baro 1500 --json
```

## Проверка

Через CTest:

```bash
ctest --test-dir build --output-on-failure
```

Напрямую:

```bash
./build/krot_cpp_tests
```

Тесты проверяют:

- разбор NMEA-строки;
- формулу `1500 - расстояние до земли`;
- высокое совпадение одинаковых профилей;
- низкое совпадение разных профилей;
- контрольный журнал на контрольной карте;
- снижение статуса на плоском рельефе.

## Ограничения

- Это технический прототип расчётной части, не готовый серийный бортовой модуль.
- Карта высот загружается из локального JSON, потоковый тайловый DEM не реализован.
- Сэмплирование карты сделано по ближайшей ячейке без сложной интерполяции.
- C++ ядро не управляет автопилотом и не содержит отказоустойчивую полётную логику.
- Реальная точность должна проверяться на заказных журналах радиовысотомера и актуальных картах района.
