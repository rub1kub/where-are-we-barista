# КРОТ / Rust-ядро расчёта

Это экспериментальная техническая версия расчётной части КРОТ на Rust.
Основной MVP остаётся в веб-приложении. Rust-ядро показывает, что алгоритм можно вынести в отдельный быстрый модуль без интерфейса.

Rust-ядро не содержит React-интерфейса, 3D-превью, карты в браузере и логики автопилота. Это отдельная командная программа:

```text
журнал радиовысотомера + карта высот -> координаты, скорость, направление, доверие
```

## Что принимает

- журнал радиовысотомера `.nmea` / `.txt` с сообщениями GGA;
- карту высот `data/dem-sample.json`;
- высоту борта над уровнем моря, по умолчанию `1500 м`;
- диапазон путевой скорости для перебора.

Поле высоты в GGA используется как расстояние до земли от радиовысотомера.
Профиль рельефа считается по формуле:

```text
высота рельефа = высота борта над уровнем моря - расстояние до земли
```

## Что выдаёт

- `x_m` / `y_m` — локальные координаты на карте высот;
- широту и долготу, если точка привязана к карте;
- путевую скорость;
- направление / азимут;
- коэффициент совпадения профилей;
- ошибку профиля;
- доверие к расчёту;
- статус: `VALID`, `DEGRADED`, `AMBIGUOUS`, `LOW_RELIEF`, `NO_FIX`.

В человекочитаемом выводе статусы печатаются по-русски: например, `VALID` как `МЕСТО НАЙДЕНО`.

## Подготовка данных

В репозитории хранится готовый контрольный набор:

```text
rust-core/data/dem-sample.json
rust-core/data/control-radio-altimeter.nmea
```

Карту высот можно пересобрать из TypeScript-сэмпла:

```bash
node tools/export_dem_from_ts.mjs
```

Контрольный NMEA-журнал скопирован из `examples/vanavara-success-radio-altimeter.nmea`. Это не данные заказчика, а fixture для проверки отдельного расчётного ядра.

## Запуск

```bash
cd rust-core
cargo run --release -- \
  --dem data/dem-sample.json \
  --nmea data/control-radio-altimeter.nmea \
  --baro 1500 \
  --speed-min 35 \
  --speed-max 65 \
  --speed-step 1
```

JSON-вывод:

```bash
cargo run --release -- --dem data/dem-sample.json --nmea data/control-radio-altimeter.nmea --baro 1500 --json
```

## Тесты

```bash
cd rust-core
cargo test
```

Тесты проверяют парсинг NMEA, формулу профиля, сравнение профилей, контрольный маршрут, слабый рельеф и отказ на несовместимом профиле.

## Зачем это нужно

Веб-стенд удобен для демонстрации, карты и объяснения работы алгоритма. Rust-ядро показывает технический путь к отдельному расчётному модулю для будущего бортового вычислителя: без интерфейса, без 3D и без зависимости от браузера.

Ограничение текущей версии: это технический прототип расчётного ядра, а не готовый серийный бортовой модуль. Следующий этап — проверка на реальных журналах радиовысотомера и профилирование под целевой вычислитель.
