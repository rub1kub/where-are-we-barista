# AI-база репозитория

Эта папка нужна для AI-инструментов, которые будут читать репозиторий через GitHub, Codex, Copilot или похожие агенты.

- `repository-knowledge-base.json` — машинно-читаемое описание продукта, архитектуры, команд, UX-правил и текущих ограничений.
- Корневой `AGENTS.md` — рабочие инструкции для автономных агентов.
- `.github/copilot-instructions.md` — краткие инструкции для GitHub Copilot.

Обновляйте JSON при изменении архитектуры, команд запуска, названия продукта, источников ЦМР или UX-правил.
